# Reports Page

The Reports page displays all analysis reports generated by agents.

## Report Types

Reports are organized by type:

| Type | Agent | Contents |
|------|-------|----------|
| `issue_analysis` | Issue Triage | Priority, summary, suggested labels, related issues |
| `pr_context` | PR Context | Risk level, change type, review suggestions |
| `meeting_summaries` | Meeting Summary | Action items, decisions, key topics |
| `slack_digest` | Slack Digest | Key discussions, decisions, action items |
| `insights` | Insights | Recurring patterns, tech debt signals |
| `trend_analysis` | Trend Analysis | Health score, risk areas, recommendations |

## Viewing Reports

Click a report to see:

- **Markdown view** — Formatted report with sections
- **JSON view** — Raw structured data
- **Metadata** — When it was generated, which job produced it, LLM usage

## CLI Access

```bash
# List all reports
python3 -m cli.main report list

# Filter by type
python3 -m cli.main report list --type issue_analysis

# View a specific report
python3 -m cli.main report view issue_analysis 42

# View raw JSON
python3 -m cli.main report view issue_analysis 42 --json
```
